# UX

This folder holds product UX artifacts used to plan implementation.

## What goes where
- ux/wireframes: low-fidelity layout sketches (Markdown + Mermaid)
- ux/flows: user flows and state flows (Mermaid)
- ux/prototypes: clickable prototype notes/links + screenshots (if used)
- ux/decisions: UX decisions and tradeoffs (short docs)

## Naming
YYYY-MM-DD-topic-slug.md

## Minimum bar
Every UX doc should include:
- Goal
- Primary users
- Main screens/states
- Edge cases
- Accessibility notes (if relevant)
